Maestro\Core\Task\FileTask

Manage a file

Description¶

This task can be used to manage the state of a file. Ensure file exists:

new FileTask(
    path: "README.md",
    contents: "Hello World"
)

Remove a file:

new FileTask(
    path: "README.md",
    exists: false
)

Create a directory

new FileTask(
    path: "myDirectory",
    type: "directory"
)
