# Cat

`Maestro\Core\Task\CatTask`

Publish contents of text file

## Parameters
- **path** Path to file in workspace - `string`

## Description
This task is useful for debugging. If you have a README file in your
workspace:
```text
Hello World
```
And the task:
```php
new CatTask(
   path: 'README.md'
);
```
The contents will be published at level INFO.
