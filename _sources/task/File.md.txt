# File

`Maestro\Core\Task\FileTask`

Manage a file

## Description
This task can be used to manage the state of a file.
Ensure file exists:
```php
new FileTask(
    path: "README.md",
    contents: "Hello World"
)
```
Remove a file:
```php
new FileTask(
    path: "README.md",
    exists: false
)
```
Create a directory
```php
new FileTask(
    path: "myDirectory",
    type: "directory"
)
```
