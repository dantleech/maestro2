# Parallel

`Maestro\Core\Task\ParallelTask`

Execute multiple tasks in parallel

## Parameters
- **tasks** Tasks to run in parallel - `array<array-key, Task>`

## Description
For example to checkout multiple repsoitories at the same time:
```php
new ParallelTask([
    // repository 1,
    // repository 2,
])
```
Typically you would use this task to setup pipelines in parallel:
```php
new ParallelTask([
    new SequentialTask([
      // other tasks
    ]),
    new SequentialTask([
      // other tasks
    ]),
])
```
